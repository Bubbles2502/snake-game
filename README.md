#Classic Snake Game

A retro, Nokia-style Snake game built entirely from scratch using HTML5 Canvas, CSS, and vanilla JavaScript. 

**[PLAY THE LIVE DEMO HERE](https://bubbles2502.github.io/snake-game/)**

##Features
* Nokia Retro Aesthetic: Styled with classic green and dark-green palettes.
* Responsive Controls: Playable with both Arrow Keys and W-A-S-D.
* Dynamic Scoring: Real-time score tracking that increases as you eat the apples.
* Game State Management: Includes a Start screen, Game Over screen, and Play Again functionality.

## How to Play
1. Click Start Game.
2. Use the Arrow Keys or W, A, S, D to steer the snake.
3. Eat the red apples to grow longer and increase your score.
4. Don't crash into the walls or your own tail!
